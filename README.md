## Build Instruction

```
npm install
```

## Deploy Instruction

```
gcloud app deploy
```

## Add Dependency Instruction

For application dependency:
```
npm install <package-name>
```

For development dependency:
```
npm install <package-name> --save-dev
```

## Current Scripts

Run test cases:

```
npm test
```

Run server locally:

```
npm start
```
